# Kelompok-8-Mikrokontroler

## Tema Project-Alat Siram Otomatis Mengunakan Soil Senror
dirancang untuk membantu proses penyiraman tanaman secara otomatis berdasarkan tingkat kelembapan tanah. Sistem ini menggunakan soil moisture sensor untuk mendeteksi kondisi tanah, sehingga tanaman dapat memperoleh air sesuai kebutuhan tanpa harus disiram secara manual, MQTT sebagai protokol komunikasi data. Melalui MQTT, data kelembapan tanah, status pompa, dan kondisi penyiraman dapat dikirim secara real-time ke aplikasi monitoring, seperti dashboard IoT atau smartphone. Pengguna dapat memantau kondisi tanaman dari jarak jauh melalui jaringan internet.

### Komponen Utama
-ESP32
-Soil Sensor
-Relay 1 module 5 Volt
-Power Supply(Batrai)
-Laptop (monitoring)
-Konektor Batrai
-Kabel (Male dan Female)
-Pump Air 5 Volt

#### Komponen Tambahan
-Obeng
-Ampere meter(mengecek tegangan Voltase Pin 5v esp32)
